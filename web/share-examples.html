Shareable URL Examples

Use these URL patterns to share specific views of the Hospital Shift Analysis dashboard

Monthly Statistics URLs

View April 2014 Statistics

index.html?tab=monthly&year=2014&month=04
Shows monthly statistics for all nurses in April 2014
Parameters: tab=monthly, year=2014, month=04

View January 2020 Sorted by Saved Rest Days

index.html?tab=monthly&year=2020&month=01&sortBy=savedRestDays
Shows January 2020 statistics sorted by saved rest days (highest first)
Parameters: tab=monthly, year=2020, month=01, sortBy=savedRestDays

View December 2019 Sorted by Work Days

index.html?tab=monthly&year=2019&month=12&sortBy=workDays
Shows December 2019 statistics sorted by number of work days
Parameters: tab=monthly, year=2019, month=12, sortBy=workDays

Person Details URLs

View 马磊's Complete Record

index.html?tab=person&nurse=3&nurseName=马磊
Shows complete career details for nurse 马磊 (ID: 3) across all years
Parameters: tab=person, nurse=3, nurseName=马磊

View 付伟's 2018 Record

index.html?tab=person&nurse=4&nurseName=付伟&personYear=2018
Shows 付伟's detailed records for 2018 only with monthly breakdown and calendar
Parameters: tab=person, nurse=4, nurseName=付伟, personYear=2018

View 赵蕊's 2016 Record

index.html?tab=person&nurse=1&nurseName=赵蕊&personYear=2016
Shows 赵蕊's detailed records for 2016 with calendar view
Parameters: tab=person, nurse=1, nurseName=赵蕊, personYear=2016

Overview URLs

View Overview Dashboard

index.html?tab=overview
Shows the overview dashboard with top performers, trends, and statistics
Parameters: tab=overview

URL Parameter Reference

Parameter Values Description
tab monthly, person, overview Which tab to display
year 2014-2020 Year for monthly view
month 01-12 Month for monthly view (zero-padded)
sortBy workValue, savedRestDays, workDays, nurseName Sort criteria for monthly view
nurse 1-13 Nurse ID for person view
nurseName 赵蕊, 尤嘉, 马磊, etc. Nurse name for person view
personYear 2014-2020 Year filter for person view

Usage Tips

  • Shareable Links: Copy any URL from the address bar after making selections to share that exact view
  • Bookmarks: Bookmark specific views for quick access to frequently used reports
  • Team Sharing: Share monthly analysis URLs with team members for specific periods
  • Performance Reviews: Use person detail URLs for individual nurse performance discussions
  • URL Updates: The URL automatically updates as you navigate, making it easy to share current view
